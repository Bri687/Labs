## Objectives

- I can describe the problem with using real network calls and databases in my tests
- I can state what a _Test Double_ is and how it solves that problem
- I can define what a _fake_ is and implement one in my code
- I can define what a _mock_ is and implement one in my code
- I can define what a _stub_ is and implement one in my code

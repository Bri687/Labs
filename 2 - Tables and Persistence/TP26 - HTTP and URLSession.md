## Objectives

- I can describe the basics of the HTTP networking protocol, including commonly used methods, response types, and response codes
- I can describe the different attributes of a URL and can create a URL with query items
- I can use `URLSession.shared` to perform basic HTTP methods for sending and receiving information
- I can print the results of a `URLSession` task to the console.

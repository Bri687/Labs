## Objectives

- I am able to use the Project Navigator in an Xcode project to find files
- I am able to switch between the Project Navigator and the other Navigators in the Navigator area.
- I am able to locate and use the Editor Area in an Xcode project.
- I am able to locate the Debug Area, and the Console pane and Variables pane inside it
- I am able to locate the Inspector area
- I am able to run an Xcode Project in Simulator.
- I am able to build a project, and define what it means to build a project.
- When build errors and warnings appear, I am able to locate them, view them, and, if Xcode suggests a fix, accept that suggestion.
- I am able to use breakpoints in my code and describe their purpose.
- I am able to access and navigate Xcode's Documentation.

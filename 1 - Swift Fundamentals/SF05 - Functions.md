## Objectives

- I can identify what a function is and what they do.
- I can create a function that takes input parameters and returns a value.
- I can use internal and external parameter labels, as well as omitted `_` argument labels.
- I can give parameters a default value.
- I can state what a function signature is.


## Lesson Outline

- Cover _Swift Fundamentals 2.2 - Strings_ and _Swift Fundamentals 2.3 - Functions_

Suggested Activities:
- Have students work in groups to practice writing function signatures following each of these rules, with a name that is reasonable for that type of function:
  - Write a function that takes no parameters and returns nothing.
  - Write a function that takes one parameter and returns nothing.
  - Write a function that takes no parameters and returns a value.
  - Write a function that takes one parameter and returns a value.
  - Write a function that takes two parameters and returns a value.
  - Write a function that takes three parameters and returns nothing.
  - If finished, write additional functions that go beyond these requirements.
